png이미지와 svg이미지의 차이점

  • png이미지

    모바일에서 선명하게 보이지 않는다, 만약 png로 저장한 아이콘을 모바일에서 선명하게 보이게 하려면 2배이상의 사이즈로 저장해야한다.

  • svg이미지

    아이콘이나 로고같은 일러스트에서 제작한 이미지만 저장이 가능합니다. (사진불가), 특징으로는 모바일에서 선명하게 보이고 확대해도 이미지가 깨지지않습니다

SVG를 이미지태그로 코딩한 스타일

아이콘을 이미지태그로 코딩해서 마우스를 오버했을때 이미지를 교체해야하는 디자인이라면 구현이 불가능 합니다.

si업체 솔루션 개발이나 어플리케이션 개발에서는 많이 활용되는 방법(홈페이지는 인기가 적음)
아이콘 폰트를 만들어서 코드를 입력하면 아이콘이 나타나는 방식
용량이 비교적 적음 그래서 많이 씀

  • 0. 구글아이콘에서 아이콘을 세팅하고(두깨나 스타일) 아이콘 아무거나 클릭 (오른쪽에 코드가 나옴)
  • 1. 오른쪽 패널에서 Static icon font 복사해서 넣어준다(계속 넣을필요 x 한번만 쓰면됨)
  • 2. 실제 홈페이지에서 사용할 아이콘을 찾는다
  • 3. 그 아이콘의 Inserting the icon 하단의 코드를 복사합니다.
  • 4. 그걸 아이콘이 들어갈 위치에 붙여넣는다
  • 5. css에서 크기와 색상 변경 가능
